# TD4_01_RhythmActionGame

## 概要

TD4（チームデザイン4）の授業課題として制作されたリズムアクションゲームです。

## 開発環境

- Visual Studio（C++）

## 操作方法

各レーンに対応するキーを押して、降ってくるノーツに合わせてタイミングよく入力してください。

## ビルド方法

1. Visual Studio でソリューションファイル（`.sln`）を開く
2. ビルド構成を `Debug` または `Release` に設定する
3. `Ctrl + Shift + B` でビルドを実行する

## ライセンス

本プロジェクトは学習目的で作成されています。
